Frequently Asked Questions

What file formats are supported?

The tool supports PNG format only for UML class diagrams. Also,ensure your file is clear and under 10MB.

How is my diagram graded?

Diagrams are evaluated based on semantic similarity and structural similarity, using advanced algorithms like Levenshtein distance and graph edit distance.

Yes, visit the Submission History page to review your previous submissions and their feedback.
